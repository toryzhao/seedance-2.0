# Seedance 2.0 Skill

**Comprehensive production pipeline for quad-modal AI filmmaking with Seedance 2.0 (ByteDance / Dreamina)**

This repository contains the **Seedance 2.0** skill for AI-powered filmmaking. It features a modular, high-efficiency structure optimized for AI agents, covering the full spectrum from minimal-intent directing to frame-precise choreography.

### Key Features
- **Modular Architecture:** Optimized for low token usage and targeted retrieval.
- **Trilingual Precision:** Comprehensive cinematic vocabulary in **Chinese (ZH)**, **Japanese (JA)**, and **Korean (KO)**.
- **Quad-Modal Protocol:** Full support for T2V, I2V, V2V, and R2V workflows.
- **Native Audio-Video:** Integrated sonic design and beat-sync techniques.
